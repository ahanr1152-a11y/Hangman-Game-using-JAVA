# Hangman-Game-using-JAVA
🎮 Hangman Game
A simple console-based Hangman Game developed in Java. The objective of the game is to guess the hidden word by entering one letter at a time before running out of attempts. This project demonstrates fundamental Java programming concepts and logical problem-solving.
🚀 Features
Guess letters one at a time
Limited number of attempts
Correct guesses reveal the hidden word
Incorrect guesses reduce remaining attempts
Win/Lose result displayed
Simple and interactive console interface
🛠️ Built With
Java
VS Code
Scanner Class

Variables & Data Types

Strings
