# Hangman-Game-using-JAVA
# 🎮 Hangman Game

A simple **console-based Hangman Game** built using **Java**. The objective of the game is to guess the hidden word by entering one letter at a time before running out of attempts. This project is designed to strengthen Core Java concepts and improve logical thinking.

---

# ✨ Features

- 🎯 Guess one letter at a time
- 🔤 Hidden word displayed using underscores
- ✅ Correct guesses reveal the letters
- ❌ Wrong guesses decrease the remaining attempts
- 🏆 Win when the complete word is guessed
- 💀 Game Over when all attempts are exhausted
- 💻 Interactive console interface
